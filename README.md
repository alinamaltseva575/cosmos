# Cosmos API 🚀

REST API для управления космическими объектами: планеты и галактики.

## Функционал
- 🔐 JWT аутентификация
- 👑 CRUD операции для администратора
- 🪐 Управление планетами
- 🌌 Управление галактиками
- 📦 PostgreSQL база данных
- 🐳 Docker контейнеризация

```bash
git clone https://github.com/ваш-username/cosmos-api.git
cd cosmos-api
